# V2 Planner Direction Conversation

## Summary
The app direction was clarified from “better progress view” into a full planning-and-adherence product.

Key outcomes:
- the app should work on both phone and desktop
- the home should be calendar-first
- multiple pages are acceptable
- strength planning should support reusable multi-week phase patterns
- run and sprint planning can start manual
- review should be a separate page
- weekly adherence such as `5/6` is a core metric
- advanced behavior should live behind geek / coach mode
- product decisions and important conversations must be documented in the repo
